# Surgical-preview paper

IEEE conference paper (IEEEtran) benchmarking off-the-shelf image-editing models for
region-confined, identity-preserving cosmetic-surgery previews.

Every number, table, and plot in the paper is generated by `scripts/` from local
experiment data — nothing is hand-typed. Generated artifacts (`tables/*.tex`,
`figures/generated/*`, `data/canonical_rows.csv`) are committed so the LaTeX project
is self-contained (Overleaf needs no Python).

## Build

```sh
make numbers    # aggregate run CSVs -> tables/*.tex + data/canonical_rows.csv
make figures    # matplotlib + PIL   -> figures/generated/*
make pdf        # latexmk -pdf main.tex
make all
```

Python deps come from `pyproject.toml` (`uv sync` once, or let `uv run` resolve them).
The scripts read the experiment repo read-only; point them elsewhere with the
`PLASTYVUE_POC` environment variable (see `scripts/config.py`).

## Overleaf

Zip the repo (or `git archive -o paper.zip HEAD`) and use New Project → Upload Project.
Compiler: pdfLaTeX, main file: `main.tex`. Python files are ignored by Overleaf; the
committed generated artifacts make the project compile as-is.
